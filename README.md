# RAG PDF Question Answering Assistant

A Streamlit application that will answer questions using content retrieved from an uploaded PDF.

## Milestone 1

This milestone provides the project structure and a runnable PDF upload interface. PDF processing and RAG are intentionally not implemented yet.

## Run locally

```powershell
python -m venv .venv
.venv\Scripts\Activate.ps1
pip install -r requirements.txt
streamlit run app.py
```
